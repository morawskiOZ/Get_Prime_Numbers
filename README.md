## General info

App where user specify the input which is called upon REST API to get
prime numbers (AJAX request) in specific range (provided by user) .


## Technologies

Project created with:
* Node
* Express
* Pug
* Webpack
* Axios


## Setup

Simply go to:   
<https://where-are-my-primes.herokuapp.com>
